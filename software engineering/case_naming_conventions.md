# Case naming conventions

## Snake Case
naming variables:
    -var_name
naming constans:
    -VAR_NAME

## Camel Case
naming variables:
    -varName
naming constans:
> We use snake upper case to define the name for constans
    -VAR_NAME

## Pascal Case
naming variables:
    -VarName
naming constans:
> We use snake upper case to define the name for constans
    -VAR_NAME
